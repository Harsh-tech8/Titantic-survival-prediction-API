# SMS Spam Detector App

SMS Spam Detector is a Machine Learning and Natural Language Processing (NLP) based web application designed to classify SMS messages as either **Spam** or **Ham (Not Spam)**. The project demonstrates an end-to-end machine learning workflow, including data preprocessing, feature extraction, model training, evaluation, and deployment.

The application uses text vectorization techniques to convert SMS messages into numerical features that can be analyzed by a trained machine learning model. Users can enter a message through a simple web interface, and the system instantly predicts whether the message is spam or legitimate.

This project is built using **Python**, **Scikit-learn**, **Flask**, and **Jupyter Notebook**. It includes pre-trained model files for efficient prediction and a lightweight web interface for easy interaction. The repository serves as a practical example of applying NLP techniques to solve real-world problems and is a valuable learning resource for developers interested in machine learning, text classification, and model deployment.

Future enhancements may include advanced deep learning models, cloud deployment, multilingual support, and improved user experience.
